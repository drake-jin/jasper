## Dockerfile Build

Dockerfile은 {명령} {매개변수} 형식으로 작성합니다. 명령은 대소문자를 구분하지 않지만
보통 대문자로 작성한다.

- \# 은 주석입니다.

Dockerfile로 빌드할 때에는 

``` bash 
$ sudo docker build --tag {image_name} {Dockerfiles dir}

```
